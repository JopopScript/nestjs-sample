# nestjs 테스트코드 작성 샘플

## 패키지 설치

```
$ npm install
```

## 실행

실행 후 localhost:3000으로 접속

### 로컬 실행

```bash
# build
$ npm run build

# production
$ npm run start

# watch mode
$ npm run start:dev
```

### docker 실행

필요한 프로그램: docker, docker-compose

```bash
# docker-compose build
# docker-compose up
```

## 테스트

```bash
# unit test
$ npm run test

# test coverage
$ npm run test:cov
```
