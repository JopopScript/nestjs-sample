# nestjs 테스트코드 작성 샘플

## 패키지 설치

```
$ npm install
```

## 실행

- localhost:3000 접속
- localhost:3000/api 스웨거문서 접속

### 로컬 실행

```bash
# build
$ npm run build

# production
$ npm run start

# watch mode
$ npm run start:dev
```

### docker 실행

필요한 프로그램: docker, docker-compose

```bash
# docker-compose build
# docker-compose up
```

## 테스트

```bash
# unit test
$ npm run test

# test coverage
$ npm run test:cov
```
